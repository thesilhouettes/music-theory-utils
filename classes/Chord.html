Class Chord

Represents a chord, which means notes are stacked together vertically. In music-theory-utils, a chord is represented by a base note, and the intervals.

see

Note and Interval for details on the base note and interval

constructor

  • new Chord(note: Note, intervals: Interval[]): Chord
    • Defined in src/types/Chord.ts:144

    Constructs a chord from the root and the corresponding intervals.

    example
    new Chord(new Note("A"), [new Interval("M3"), new Interval("m3")]);
    // "A C# E" (A major triad)

    new Chord(new Note("C", "b"), Chord.minorSeventh);
    // "Cb Ebb Gb Bbb"

    Parameters

    • note: Note

      the root

    • intervals: Interval[]

      the intervals which make up the chord, see the examples below.

    Returns Chord
