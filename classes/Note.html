difference

  • difference(rhs: Note): number
    • Defined in src/types/Note.ts:269

    Calculate the number of half steps between the current note and an input note. The number is counted like this: the first note is exclusive, while the second note is inclusive.

    example
    new Note("C").difference(new Note("A")) // 9
    remarks

    may return negative values if the first note is lower than the second note, also applies for relative notes

    throws

    NotSameTypeError if the input notes do not have the same type

    Parameters

    • rhs: Note

      another note

    Returns number

    an integer representing the number of half steps between two notes

equals

  • equals(rhs: Note, enharmonicallyEquivalent?: boolean): boolean
    • Defined in src/types/Note.ts:540

    Check if two notes are equal.

    Two notes are said to be equal if they have:

    • the same pitch
    • the same octave (or same type)
    • the same quality if enharmonicallyEquivalent is not set to true. If enharmonicallyEquivalent is set to true, then two notes are considered true if they sound the same (that is, the position of them is identical)

    Parameters

    • rhs: Note

      another note to compare

    • enharmonicallyEquivalent: boolean = false

      return the same if the two notes sound the same

    Returns boolean

    true if they are equal, false otherwise
